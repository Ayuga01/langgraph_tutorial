# LangGraph Tutorial Repository

This repository contains my hands-on learning and experimentation with **LangGraph**, a framework for building stateful, multi-step AI applications using LLMs.

## About

The purpose of this repository is to:
- Understand how LangGraph works internally
- Build and test graph-based LLM workflows
- Explore state management in AI pipelines
- Experiment with real-world use cases like agents and chains

This is a **learning-focused repository** documenting my progress with LangGraph.

---

## What is LangGraph?

LangGraph is built on top of LangChain and allows developers to:
- Create graph-based workflows instead of linear chains
- Maintain state across multiple steps
- Build complex agent systems with branching logic
